# Recommend Learning Resources for Unity

There are many videos, blog posts, courses and training material out there for Unity game developers. The problem is that with so many different Unity versions and core package versions you often find things that are no longer accurate and outdated. Additionally it can just be an overwhelming amount of information. Here is a collection of some of the best sources for learning game development with Unity.

## Unity Provided

* [Unity Learn](https://learn.unity.com)
* [Unity YouTube](https://www.youtube.com/c/unity)
* [Unity Official Discord](https://discord.com/invite/Unity)
* [Unity Answers](https://answers.unity.com)

## Udemy Courses

* [Complete C# Unity Game Developer 2D](https://www.udemy.com/course/unitycourse/)
* [Complete C# Unity Game Developer 3D](https://www.udemy.com/course/unitycourse2/)
* [Unity Tech Art: Realistic Lighting For Game Development](https://www.udemy.com/course/lighting-in-unity/)
* [The Ultimate Guide to Game Development with Unity - Official](https://www.udemy.com/course/the-ultimate-guide-to-game-development-with-unity/)
